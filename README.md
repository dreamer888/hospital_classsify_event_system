# 医院等级评审与事件管理系统

#### 介绍
医院评审是指医院按照本办法要求，根据医疗机构基本标准和医院评审标准，开展自我评价， 持续改进医院工作，并接受卫生行政部门对其规划级别的功能任务完成情况进行评价，以确定医院等级的过程，每年都需要进行评审。

不良事件管理是医疗质量安全的核心内容,医院应当建立医疗质量(安全)不良事件管理制度,并通过对医疗质量(安全)不良事件的分析,促进医疗质量提高。
